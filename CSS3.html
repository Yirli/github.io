CSS 3

Align Content

Especifíca la alineación entre las líneas escritas dentro de un contenedor flexible.
La lista de valores válidos corresponde a: stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit
Ejemplo: div{align-content: center}

Align Self

Especifíca la alineación para los ítems seleccionados dentro de un contenedor flexible.
La lista de valores válidos corresponde a: auto|stretch|center|flex-start|flex-end|baseline|initial|inherit
Ejemplo: div{align-self: center}

Border Width

Especifica el ancho de los cuatro bordes
La lista de valores válidos corresponde a: medium|thin|thick|length|initial|inherit
Ejemplo: div {border-width: thin;}

Color

Especifica el color del texto
La lista de valores válidos corresponde a: color|initial|inherit
Ejemplo: div {color: red;}

Caption Side

Especifica la colocacion de la leyenda de una tabla
La lista de valores válidos corresponde a: top|bottom|initial|inherit
Ejemplo: div {caption-side: bottom;}

Column Span

Especifica cuantas columnas debe abarcar un elemento
La lista de valores válidos corresponde a: none|all|initial|inherit
Ejemplo: h2 {column-span: all;}

Column Width

Especifica el ancho de la columna
La lista de valores válidos corresponde a: auto|length|initial|inherit
Ejemplo: h2 {column-width: 100px;}

Font Family

Especifica la fuente de texto para el elemento
La lista de valores válidos corresponde a: family-name|generic-family|initial|inherit
Ejemplo: p{font-family: Arial, Helvetica, sans-serif;}

Font Size

Especifica el tamaño del texto para el elemento
La lista de valores válidos corresponde a: medium|xx-small|x-small|small|large|x-large|xx-large|smaller|larger|length|initial|inherit
Ejemplo: div {font-size: 15px;}

Font Style

Aplica estilos al texto del elemento
La lista de valores válidos corresponde a: normal|italic|oblique|initial|inherit
Ejemplo: div {font-style: italic;}

Height

Establece la altura de un elemento
La lista de valores válidos corresponde a: auto|length|initial|inherit
Ejemplo: div {height: 100px;}

Letter Spacing

Establece el espacio entre caracteres en un texto
La lista de valores válidos corresponde a: normal|length|initial|inherit
Ejemplo: div {letter-spacing: 3px;}

Margin

Establece los margenes para los cuatro lados de un elemento en el orden: top, right, bottom, left
La lista de valores válidos corresponde a: length|auto|initial|inherit
Ejemplo: div {margin: 10px 5px 15px 20px;}

Opacity

Especifica el nivel de opacidad de un elemento, donde 0 es completamente transparente, el valor por defecto es 1
La lista de valores válidos corresponde a: number|initial|inherit
Ejemplo: h2 {opacity: 0.5;}

Padding

Establece el espacio entre el contenido y el borde del elemento para los cuatro lados en el orden: top, right, bottom, left
La lista de valores válidos corresponde a: length|initial|inherit
Ejemplo: div {padding: 35px 70px 50px 90px;}

Text Align

Establece la alineacion horizontal del texto
La lista de valores válidos corresponde a: left|right|center|justify|initial|inherit
Ejemplo: div {text-align: center;}

User Select

Establece si el texto de un elemento puede ser selecionado o no
La lista de valores válidos corresponde a: auto|none|text|all
Ejemplo: div {user-select: none;}

Vertical Align

Establece la alineacion vertical de un elemento
La lista de valores válidos corresponde a: baseline|length|sub|super|top|text-top|middle|bottom|text-bottom|initial|inherit
Ejemplo: div {vertical-align: inherit;}

Visibility

Establece la visibilidad de un elemento
La lista de valores válidos corresponde a: visible|hidden|collapse|initial|inherit
Ejemplo: div {height: 100px;}

Width

Establece el ancho de un elemento
La lista de valores válidos corresponde a: auto|value|initial|inherit
Ejemplo: div {width: 100px;}
