HTML 5

Object

Esta etiqueta define un objecto empotrado dentro de un documento HTML. Es utilizado para colocar elementos multimedia en la página, así como otras páginas web. Su sintaxis corresponde a "< object> < / object>"
Atributos:
type, name, height, form, data
Ejemplo:
< object width="400" height="400" data="helloworld.swf">< /object>

Article

Representa una composición completa de un documento. Podría referirse a un post en un foro, un artículo en una revista o periódico. Su sintaxis corresponde a "< article> < / article>"
Atributos:
display
Ejemplo:
< article itemscope itemtype="http://schema.org/BlogPosting">

Aside

Define otro contenido a parte del contenido que ya está definido donde se encuentra este. Su sintaxis corresponde a "< aside> < / aside>"
Atributos:
Soporta los atributos globales y de eventos de HTML
Ejemplo:
< aside>
< h4>Epcot Center< /h4>
< p>The Epcot Center is a theme park in Disney World, Florida.< /p>
< /aside>

Header

Este elemento representa un contenedor para contenido introductorio o un conjunto de links. Su sintaxis corresponde a "< Header> < / Header>"
Atributos:
Soporta los atributos globables y de eventos de HTML
Ejemplo:
< header>
< h1>Most important heading here< /h1>
< /header>

Tbody

Corresponde a un pequeño programa, usualmente en java, que se pueden insertar en una página. Su sintaxis corresponde a "< applet> < / applet>"
Atributos:
archive, codebase, height, width, name
Ejemplo:
< tbody>
< tr>
< td>January< /td>
< td>$100< /td>
< /tr>
< /tbody>

Nav

Esta etiqueta define un conjunto de links. Su sintaxis corresponde a "< nav> < / nav>"
Atributos:
Soporta los atributos globables y de eventos de HTML
Ejemplo:
< nav>
< a href="/html/">HTML< /a>< /nav>

Audio

Permite insertar algún sonido como música a la página web. Acepta los siguientes formatos: MP#, WAV, OGG. Su sintaxis corresponde a "< audio> < / audio>"
Atributos:
autoplay, controls, loop, muted, src, preload
Ejemplo:
< audio controls>
< source src="horse.ogg" type="audio/ogg">
< /audio>

Canvas

Esta etiqueta es utilizada para mostrar gráficos por medio de JavaScript. Esta actúa únicamente como un contenedor de gráficos. Su sintaxis corresponde a "< canva>< / canva> "
Atributos:
height, width
Ejemplo:
< canvas id="myCanvas">< /canvas>

A

Esta etiqueta define un hipervínculo, el cual es utilizado para ir de una página a otra. Su sintaxis corresponde a "< a>< / a> "
Atributos:
href, download, hreflang, media, type
Ejemplo:
< a href="https://www.w3schools.com">Visit W3Schools.com!< /a>

Link

Define un link entre un documento y otro archivo externo, por ejemplo es utilizado para agregar las hojas de estilo al documento HTML. Su sintaxis corresponde a "< link>"
Atributos:
href, media, rel, hreflang, sizes, type
Ejemplo:
< link rel="stylesheet" type="text/css" href="theme.css">

IFrame

Específica un inline frame, el cual es utilizado para empotrar otro documento dentro de documento HTML actual. Su sintaxis corresponde a "< iframe>< /iframe>"
Atributos:
height, name, sandbox, src, width
Ejemplo:
< iframe src="https://www.w3schools.com">< /iframe>

Select

Se utiliza para creat una lista "drop-down". Mediante el uso de la etiqueta option, se definen los elementos dentro de esta lista. Su sintaxis corresponde a "< select>< / select> "
Atributos:
autofocus, disable, name, multiple, required, size
Ejemplo:
< select>
< option value="volvo">Volvo< /option>
< / select>

Button

Esta etiqueta define un botón clickleable. Dentro de este se pueden colocar imagenes o texto. Su sintaxis corresponde a "< button>< /button> "
Atributos:
autofocus, disabel, form, formmethod, name, target
Ejemplo: < button type="button">Click Me!< /button>

Label

Permite definir un texto para un botón, área de texto, un select, entre otros. Su sintaxis corresponde a "< label>< / label> "
Atributos:
for, form
Ejemplo:
< label for="male">Male

Caption

Esta etiqueta define una leyenda a una tabla. Debe colocarse inmediatamente después de la etiqueta table. Su sintaxis corresponde a "< caption>< / caption> "
Atributos:
align
Ejemplo:
< caption>Monthly savings< /caption>
