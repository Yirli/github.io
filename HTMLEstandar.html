HTML Estándar

Body

Es una etiqueta de inicio. Se utiliza para encerrar todo el cuerpo de la página. Su sintaxis corresponde a "< body> < / body>"
Atributos:
onafterprint, onbeforeunload, onmessageerror, onlanguagechange, onpopstate
Ejemplo:
< body ononline="update(true)" onoffline="update(false)" onload="update(navigator.onLine)"> < /body>

Article

Representa una composición completa de un documento. Podría referirse a un post en un foro, un artículo en una revista o periódico. Su sintaxis corresponde a "< article> < / article>"
Atributos:
itemscope, itemtype, is
Ejemplo:
< article itemscope itemtype="http://schema.org/BlogPosting">

H1...H6

Representan los encabezados para las diferentes secciones. Estos tienen un rank, en el cual el H1 es el mayor mientras que el H6 es el menor. Su sintaxis corresponde a "< Hk> < / Hk>"
Atributos:
color, font-size, text align,
Ejemplo:
< h1>Title number one< /h1>

Footer

Representan el pie de página para las diferentes secciones. Estos tienen un rank, en el cual el H1 es el mayor mientras que el H6 es el menor. Su sintaxis corresponde a "< Hk> < / Hk>"
Atributos:
color, font-size, text align, display,draggable
Ejemplo:
< footer>Footer number 1< /footer>

Applet

Corresponde a un pequeño programa, usualmente en java, que se pueden insertar en una página. Su sintaxis corresponde a "< applet> < / applet>"
Atributos:
archive, codebase, height, width, name
Ejemplo:
< applet code="Bubbles.class" width="350" height="350"> Java applet that draws animated bubbles.< /applet>

Script

Esta etiqueta se utiliza para definir el cliente de JavaScript. Puede contener los scripts o apuntar a un documento externo que tenga estos. Su sintaxis corresponde a "< script> < / script>"
Atributos:
src, type, async, charset, defer
Ejemplo:
< script> document.getElementById("demo").innerHTML = "Hello JavaScript!"; < /script>

Br

Inserta una nueva línea dentro de un texto. Esta es una etiqueta vacía, por lo que no requiere una etiqueta de cierre. Su sintaxis corresponde a "< br> "
Atributos:
class, hidden, id, lang, style
Ejemplo:
This text contains< br> a line break.

Embed

Esta etiqueta representa un contenedor para aplicaciones externas o contenido interactivo. Su sintaxis corresponde a "< embed> "
Atributos:
height, src, type, width
Ejemplo:
< embed src = "/html/yourfile.mid" width = "250" height = "100" />

Dir

Esta etiqueta es usada para especificar una lista de directorios. Es muy similar a la etiqueta ul. Su sintaxis corresponde a "< dir>< / dir> "
Atributos:
height, src, type, width
Ejemplo:
< dir>
< li>dir< /li>
< li>menu< /li>
< li>ul< /li>
< /dir>

Comment

Permite al programador poner un comentario en su código HTML. Esta es una alternativa para la etiqueta < !--....-->, sin embargo solo es soportada por IE. Su sintaxis corresponde a "< comment>< / comment> "
Atributos:
No cuenta con atributos específicos
Ejemplo:
< comment>This is a commented line in IE< /comment>

Hr

Se utiliza para crear una línea horizontal. Es también llamada "Horizontal Rule". Su sintaxis corresponde a "< hr />"
Atributos:
align, nonshade, size, width
Ejemplo:
< p>This text will be followed by a horizontal line < hr />< /p>

Layer

Se utiliza para posicionar y animar elementos en la página. Puede ser pensada como un documento aparte que se encuentra sobre el principal, todos coexistiendo en la misma página. Su sintaxis corresponde a "< layer>< / layer> "
Atributos:
above, background, below, bgcolor, clip, src, top
Ejemplo:
< layer id = "layer1" top = "250" left = "50" width = "200" height = "200" bgcolor = "red">
< p>layer 1< /p>
< /layer>

Option

Esta etiqueta es utilizada dentro de un menú de opciones para definir estas en una lista "drop-down" para que el usuario pueda seleccionar la opción que prefiere. Su sintaxis corresponde a "< option>< / option> "
Atributos:
disabled, label, selected, value
Ejemplo:
< option value = "Java" selected>Java

Map

Permite definir una imagen de un mapa al ser utilizada con la etiqueta img. Su sintaxis corresponde a "< map>< / map> "
Atributos:
name
Ejemplo:
< map name = "html">
< area shape = "circle" coords = "154,150,59" href = "about/about_team.htm" alt = "Team" target = "_self" />
< /map>

Text Area

Esta es utilizada dentro de un formulario para declarar un elemento de texto, donde el usuario pueda escribir utilizando múltiples líneas. Su sintaxis corresponde a "< textarea>< / textarea> "
Atributos:
autofocus, cols, disabled, form, maxlength, placeholder, rows, wrap
Ejemplo:
< textarea rows = "5" cols = "50" name = "description">
Enter your name
< /textarea>

Table

Esta etiqueta es utilizada para definir una tabla. Dentro de esta se encuentran otras etiquetas para definir la estructura de una tabla. Su sintaxis corresponde a "< table>< / table> "
Atributos:
align, bgcolor, border, cellpadding, frame, abbr
Ejemplo:
< table border = "1">
< tr>
< th>Team< /th>
< th>Ranking< /th>
< /tr>
